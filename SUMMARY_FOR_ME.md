# 청년도약계좌 vs 청년미래적금 계산 페이지 요약

이 폴더는 청년도약계좌를 계속 유지할지, 2026-07-27 특별중도해지 후 2026-07-28 청년미래적금으로 전환할지 비교하는 정적 웹페이지 프로젝트입니다.

## 어디까지 됐는지

- `index.html` 페이지와 `src` 폴더의 계산/화면 코드가 만들어졌습니다.
- 청년도약계좌 기여금, 청년미래적금 기여금, 외부 돈 세전→세후 이율, 납입 한도 검증이 들어가 있습니다.
- 납입 이력 자동 생성은 매년 가입월에 가입일 이후부터 납입 가능하도록 반영했습니다. 예: 2024-06-29 가입이면 2025년 6월 납입은 2025-06-29로 생성됩니다.
- 해지일이 그 달 납입 가능일보다 빠르면 해당 월 납입은 빠집니다. 예: 2024-07-29 가입이고 2026-07-27 해지면 2026년 7월 납입은 생성되지 않습니다.
- 납입 이력을 직접 수정하기 전에는 가입일/해지일/입금일/기본금액을 바꾸면 납입 이력이 자동으로 다시 생성됩니다.
- 연 840만원 한도는 달력연도가 아니라 가입일 기준 납입연도로 검증합니다.
- 입력값, 선택 은행, 현재 탭, 납입 이력은 브라우저에 자동 저장되어 새로고침 후에도 유지됩니다. `저장값 초기화` 버튼으로 기본값으로 되돌릴 수 있습니다.
- 청년미래적금 은행별 우대금리 확인 탭이 있고, 가입은행을 고르면 해당 은행의 요약뿐 아니라 상세조건, 만기 후 금리, 유의사항까지 보여줍니다.
- 청년도약계좌 유지 시 1~36개월 고정금리, 37~48개월 변동금리 예상값, 49~60개월 변동금리 예상값을 따로 입력할 수 있습니다. 원금 이자와 기여금 이자 모두 이 구간별 금리로 나눠 계산합니다.
- `사용된 논리` 탭에서 납입일 보정, 월/연 한도, 2026년 7월 기여금 처리, 금리 스케줄, 외부 현금 계산 순서 등을 확인할 수 있습니다.
- 단위 테스트는 `npm.cmd test`로 실행합니다.

## 직접 실행하려면

VSCode에서 폴더를 열고 터미널에서 아래를 실행한 뒤 브라우저로 접속하면 됩니다.

```powershell
python -m http.server 4173 --bind 127.0.0.1
```

주소:

```text
http://127.0.0.1:4173/
```

## 수정할 만한 값

- 화면의 가입일, 입금일, 납입금액, 금리, 외부 돈 세전 연이율을 직접 바꾸면 바로 재계산됩니다.
- 도약 만기 금리와 도약 기여금 이자 금리는 1~36개월, 37~48개월, 49~60개월로 나뉘어 있습니다. 기본값은 모두 4.5%입니다.
- 납입 이력 탭에서 각 납입일과 금액을 수정하거나 행을 추가/삭제할 수 있습니다.
- 은행별 우대금리 데이터는 `src/data.js`에 들어 있습니다. 공시가 바뀌면 이 파일을 갱신해야 합니다.

## 주의사항

- 저장값은 GitHub Pages 서버가 아니라 사용자의 브라우저 `localStorage`에 저장됩니다. 같은 브라우저/같은 주소에서는 유지되지만, 다른 브라우저나 시크릿 창에는 공유되지 않습니다.
- 외부 돈 이율만 15.4% 과세 후 세후 이율로 계산합니다.
- 청년도약계좌와 청년미래적금 상품 이자는 비과세 전제로 계산합니다.
- 실제 은행의 특별중도해지 우대금리 적용 여부는 은행 답변과 달라질 수 있으므로, 페이지에서 금리를 직접 조정해서 비교해야 합니다.
