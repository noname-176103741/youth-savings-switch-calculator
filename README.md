# 청년도약계좌 vs 청년미래적금 전환 계산기

청년도약계좌를 계속 유지하는 경우와, 2026-07-27 특별중도해지 후 2026-07-28 청년미래적금으로 전환하는 경우를 비교하는 계산기입니다.

별도 서버나 백엔드 없이 브라우저에서 바로 실행되며, 입력값은 사용하는 브라우저의 localStorage에 저장됩니다.

## 주요 기능

- 청년도약계좌 특별중도해지 금리, 만기 금리, 기여금 이자 금리 입력
- 청년도약계좌 1-36개월 고정금리, 37-48개월/49-60개월 변동금리 예상값 분리 입력
- 청년미래적금 금리와 정부기여금 유형 선택
- 청년미래적금 납입금 적용 금리와 정부기여금 이자 금리 분리 입력
- 청년도약계좌 가입일, 해지일, 입금일, 납입 이력 자동 생성 및 수동 편집
- 청년도약계좌 월 70만원, 가입일 기준 납입연도 840만원 한도 검증
- 청년도약계좌 소득구간 1-5년차 선택
- 청년미래적금 은행별 우대금리 요약/상세조건 조회
- 외부 돈 세전 연이율 입력 시 15.4% 과세 후 세후 연이율로 계산
- 전환/유지 비교 결과, 추가 납입필요, 손익분기 외부 연이율 표시
- 브라우저 localStorage 기반 입력값 자동 저장
- `사용된 논리` 탭에서 계산 가정과 규칙 확인

## 실행 방법

정적 파일이므로 `index.html`을 브라우저에서 열어도 동작합니다.

로컬 서버로 확인하려면:

```powershell
python -m http.server 4173 --bind 127.0.0.1
```

브라우저에서 아래 주소를 엽니다.

```text
http://127.0.0.1:4173/
```

## 계산 기준 요약

- 청년도약계좌 상품 이자와 청년미래적금 상품 이자는 비과세로 봅니다.
- 외부 현금 계정의 이자만 15.4% 세금을 차감한 세후 수익률로 계산합니다.
- 저장값이 없을 때 초기 금리 기본값은 도약 특별중도해지 4.5%, 도약 만기 6%/5.5%/5.5%, 도약 기여금 이자 4.5%/4%/4%, 미래적금 8%, 미래적금 기여금 이자 5%, 외부 돈 세전 4.5%입니다.
- 청년도약계좌 유지 시 원금 이자와 기여금 이자는 1-36개월, 37-48개월, 49-60개월 금리 구간으로 나누어 단리 일할 계산합니다.
- 정부기여금은 월 납입분 기준 다음 달 10일 정산으로 처리합니다.
- 청년도약계좌 2026년 7월 납입분 기여금은 받는 것으로 보되, 해지일 이후 정산되어 이자는 붙지 않는 것으로 처리합니다.
- 청년도약계좌 소득구간 1년차는 가입일부터 다음 해 가입월 말일까지, 2년차부터는 매년 가입월 다음 달 1일부터 적용합니다.
- 청년미래적금 월 납입액은 월 50만원 한도를 적용합니다.
- 청년미래적금 정부기여금 이자는 납입금 적용 금리와 분리해 기본값 연 5.0%로 계산합니다.
- 납입일과 한도 검증의 세부 규칙은 앱의 `사용된 논리` 탭에서 확인할 수 있습니다.

## 주의사항

- 이 계산기는 개인 비교용 보조 도구입니다.
- 실제 금리, 우대금리 충족 여부, 특별중도해지 적용 금리, 정부기여금 적용 여부는 가입 은행과 공식 안내를 반드시 확인해야 합니다.
- 은행별 공시는 변경될 수 있으며, 이 저장소의 정적 데이터와 최신 공시 사이에 차이가 있을 수 있습니다.
- 브라우저에 저장되는 입력값은 서버가 아니라 해당 브라우저의 localStorage에만 저장됩니다.
